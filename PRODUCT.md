## Product: AttendSure

A simple web app for HR attendance integrity + SOP management.

### Users & roles
- **HR (admin):** uploads SOP/rules documents; views the SLA / attendance-validity dashboard across employees.
- **Employee:** views applicable SOPs/rules; submits a face capture to record attendance.

### Core workflow (the verified operation)
A **verified attendance check-in**: an employee submits a face capture that the system confirms is a real human face, recording a valid attendance for that day. The system validates the capture: real human face = **valid**; not a human face = **invalid** (binary, explicit). This removes buddy-punching fraud.

### Features
- **Face-verified check-in** (employee): camera capture, system validates human vs non-human, records valid/invalid attendance for the day.
- **SOP / rules library** (HR uploads, employees view): one authoritative place for policy documents.
- **SLA / compliance dashboard** (HR): at-a-glance view of whether each employee validly attended — turning raw check-ins into a clear attendance-integrity signal.

### Brand & tone
Trustworthy, clean, minimal, fast. The product makes attendance validity obvious and credible. Simple UI is a hard requirement on both roles.

### Strategic principles
- Day-one focus: trustworthy face-validated attendance + an HR dashboard that makes validity obvious.
- SOP upload supports the attendance integrity goal, not the other way around.
- Validation is binary and explicit.
